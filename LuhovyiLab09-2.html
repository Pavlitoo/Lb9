Розділ 1: Вступ до веб-технологій

HTML (HyperText Markup Language) - це мова розмітки, яка використовується для створення веб-сторінок. Вона визначає структуру та вміст документа.

Важливо: Завжди використовуйте семантичні теги для кращої доступності.

Розділ 2: Основи CSS

CSS (Cascading Style Sheets) дозволяє стилізувати HTML елементи, додаючи кольори, шрифти, відступи та багато іншого.

"Дизайн - це не просто те, як це виглядає. Дизайн - це те, як це працює." - Стів Джобс

Розділ 3: Класи та їх застосування

Класи в CSS дозволяють застосовувати один і той самий стиль до різних елементів без дублювання коду.

Використовуйте класи для повторюваних стилів та ID для унікальних елементів.

У цьому документі використовується кілька класів: section-title, content-text, note, important, та quote.

Кожен клас можна застосувати до будь-якого елемента кілька разів в одному документі.
